# Sass、LESS是什么？⼤家为什么要使⽤他们？

```
他们是CSS预处理器。他是CSS上的⼀种抽象层。他们是⼀种特殊的语法/语⾔编译成CSS。
例如Less是⼀种动态样式语⾔. 将CSS赋予了动态语⾔的特性，如变量，继承，运算， 函数. LESS 既可
以在客⼾端上运⾏ (⽀持IE 6+, Webkit, Firefox)，也可⼀在服务端运⾏ (借助 Node.js)。
为什么要使⽤它们？
结构清晰，便于扩展。
可以⽅便地屏蔽浏览器私有语法差异。这个不⽤多说，封装对浏览器语法差异的重复处理，减少⽆意
义的机械劳动。
可以轻松实现多重继承。
完全兼容 CSS 代码，可以⽅便地应⽤到⽼项⽬中。LESS 只是在 CSS 语法上做了扩展，所以⽼的 CSS
代码也可以与 LESS 代码⼀同编译。
```
